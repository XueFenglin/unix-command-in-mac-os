###  reading files

* cat
  - concatenate

* more
  - Paginated output

* less
  - backward scrolling
  - better memory use
  - less > more

example

```
less -MN filename
```
shortcut :
  b: backward
  f: forward
  q: quit
  g: begining
  shift+g : end

###  reading portions of files

* head
  - Dislplay lies from beginning of a file
* tail
  - display lines form end of a file

* tail -f
  - "Follow" the tail of a file
